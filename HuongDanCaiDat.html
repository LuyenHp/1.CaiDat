Hướng dẫn cài đặt Visual studio code và Github Git

Cài đặt Visual Studio Code

  1. Cài đặt Visual Studio Code
  2. Vào trang chủ của Visual Studio Code :https://code.visualstudio.com/download

Cài đặt Git

  1. Vào trang chủ của GIT tải về và cài đặt :https://git-scm.com/downloads

Tạo tài khoản Github

  1. Đăng nhập vào link vào tạo tài khoản: https://github.com/

Sử dụng Git trong Visual Studio Code

Sử dụng lệnh trong Cmd or PowerShell,Git để thêm tài khoản

  1. Sử dụng câu lệnh sau để cập nhật email và tên tài khoản Git.
      2. $ git config --global user.name "User Name"
      $ git config --global user.email "username@gmail.com"
  2. Các câu lệnh để tạo Repositories,Push,Clone
      3. Tạo Repositories ấn Ctr + Shift + P ,chọn Push to Github,Đặt tên cho Repositories.
      Push code lên Repositories,chọn nhánh Master ở góc dưới bên trái. Ấn Ctr +Shift+G ,Nhập mô tả , Ấn Ctr + Enter
